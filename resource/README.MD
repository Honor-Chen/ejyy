## 对象存储预存文件

### 资产导入excel

将本目录下`固定资产导入模板.xlsx`上传至对象存储`template`目录中

### 用户默认头像

将本目录下`default.png`上传至对象存储`avatar`目录中


### 导入数据库

将本目录下的`db.sql`导入`mysql`数据库
